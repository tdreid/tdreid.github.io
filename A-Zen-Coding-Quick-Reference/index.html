A Zen Coding Quick Reference

Emmet (formerly called Zen Coding ― a term I prefer for silly reasons) is a collection of plugins
for text editors that helps you write structured code like HTML and XML faster.

It’s built into a lot of modern text editors (e.g. VS Code) and easy to get
if you don’t have it already. In fact, many devs use the basics of Emmet
simply by following the code assist in their favorite IDE.

Familiarity with the full range of of abbreviations and the simple syntax
for nesting, expanding and balancing tags can unleash a lot of extra speed
when coding.

Here are the notes I keep taped to the left of my keyboard for those
moments when I need to go beyond div#page>div.logo+ul#navigation>li*5>a.
