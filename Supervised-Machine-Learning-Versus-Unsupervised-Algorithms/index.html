Supervised Machine Learning Versus Unsupervised Algorithms

Broadly speaking, machine learning algorithms fall into two categories: supervised and unsupervised. These differ according to the features of the data to be fed into the machine learning algorithm as well as the uses for the model that will be generated.



Photo courtesy [Miguel Á. Padriñán][6]

Supervised Machine Learning

When our available training data include known examples of a value that we wish to predict a supervised machine learning algorithm can be used. This will train a model to make the prediction based on the other related features (i.e. columns) in the training data.

For example, suppose we’re training a model to predict next day opening stock prices. For this task we’d provide a set of training data where each row includes many features of the stock for a given trading day. Features could include its high, low, opening, and closing prices as well as volume to mention only a few. Also each row would contain the known historical result for the value we intend to predict ― in this case the next day’s opening price.

From there, the next step is to provide a large set of of these rows to a supervised learning algorithm. There are many to choose from and research is ongoing. Three families of algorithms with which I’ve worked for stock market predictions are:

  • Linear regressions
  • Decision forests
  • Neural Networks

Our selected algorithm then does its best to discern the relationship(s) between the features we’ve provided and the next day’s opening stock price. Using the results of this analysis the algorithm creates a model that is trained to predict the price whenever new values are provided for the same features.

So, given novel inputs for high, low, open, close and volume, etc. the trained model returns a prediction for the next day’s open. It will also tell us something about the confidence in that prediction.

The accuracy of the model’s prediction depends largely on the quality and volume of the training data, the strength of the algorithm for the type of task, and how we humans are framing the questions.

Machine learning tools, including libraries of validated and optimized algorithms, are already within reach of the computing public. Accurate and interesting experiments are emerging daily on platforms provided by [Google][3], [Microsoft][4], [Amazon][5] and others.

Using supervised machine learning techniques, I’ve already trained one model to spot value stocks that should have standout price performance over the next 10 trading days.

For initial training data I used three years of daily market history combined with daily economic indicators. More than half of the evenings I devoted to designing this experiment revolved around acquiring and preparing the training set.

Even so, it only took about a week in my spare time to develop a predictive model with a [coefficient of determination][1] (R2) of nearly .997. This means almost all of the variance in the 10 day price outlook can be explained by the features as modeled.

I began trading on this bot’s predictions in early 2020. So far the results have been promising despite the harshness of the [downturn][2] associated with Coronavirus outbreak (COVID-19) in March.

Unsupervised Machine Learning

So, while supervised machine learning is about making predictions the goal of its unsupervised counterpart is to spot islands of similarity in seas of difference. That is to say unsupervised machine learning algorithms analyze data to identify clusters with shared characteristics.

To stick with stocks as our example, let’s consider all the companies traded on NYSE and NASDAQ. For each company there are prices, volume and other data for numerous trades. This kind of heterogeneous data can be passed into to an unsupervised machine learning algorithm.

The algorithm analyzes the data set to identify trades that fit certain patterns. In this case, machine learning might be used, for example, to recognize trading patterns prevalent in different segments of the economy.

In Summary

Here are some key differences between supervised machine learning and unsupervised:
| | SUPERVISED | UNSUPERVISED |
|————–|————|————–|
|Goal|Prediction |Clustering |
|Training Data|Labeled |Unlabeled|
|Utility| Forecast given new data with same features | Insights into existing data
