Define conditional CSS selectors

@import

  • Unlike standard CSS, locate @import statements in code as convenient.
  • Any file extension except .css is treated as .less.
  • options: reference (use, do not output),
    inline (include, do not process), less, css, (only import) once,
    multiple (import over and over again), optional (continue if missing)

Nesting

  • Rules nested inside another rule compile sensibly.
  • What happens in the nest stays in the nest (scoped).
  • Applies to @media queries as well.
  • & refers to all parent selectors of a nested rule.
